1. Wat is het verschil tussen het internet en het world wide web?

Het internet is het digitale netwerk waarop de meeste computers aangesloten zijn. Daarbij moet je denken aan de kabels, de glasvezels, de netwerkapparatuur, de communicatiesatellieten.

Daar kun je van alles overheen sturen. Www is een toepassing die daar gebruik van maakt. E-mail gaat ook over het internet en dat is geen www. Ook internettelefonie (bv. Skype) is geen www. En zo zijn er nog heel wat te noemen. Www is alleen maar de manier om webpagina's via links met elkaar te koppelen. Al wordt www vaak HET internet genoemd maar dat is eigenlijk fout.

Dus: internet is het fysieke netwerk en www is één van de toepassingen die je er overheen stuurt.



2. Wat is het verschil tussen HTML, CSS en JavaScript?

HTML is een uitwisselingsformaat waarmee de structuur van een website wordt beschreven.
CSS is een taal die wordt gebruikt om het design van een webpagina te beschrijven.
JavaScript is een taal die wordt gebruikt om interactieve elementen aan een webpagina toe te voegen.



3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.

HTML tags geven bepaalde informatie over de informatie die tussen de tags staat. Een voorbeeld is de tag die wordt gebruikt om een alinea mee aan te duiden. Toegepast in een pagina, zou de alinea er als volgt uit kunnen zien:



Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Aenean commodo ligula eget dolor. Aenean massa. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Donec quam felis, ultricies nec, pellentesque eu, pretium quis, sem.

Zoals je ziet, wordt een alinea eigenlijk bepaald door een paar tags: een openingstag een eindtag. Voor en eindtag wordt het symbool ‘/’ gebruikt.
