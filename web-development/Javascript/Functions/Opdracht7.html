            Wat zijn rest parameters? Geef een voorbeeld.
            De rest parameter syntax laat ons toe om een onbepaald aantal argumenten te vertegenwoordigen als een array.

            function fun1(...theArgs) {
                console.log(theArgs.length);
              }
              
              fun1();  // 0
              fun1(5); // 1
              fun1(5, 6, 7); // 3

            Wat is het verschil tussen rest parameters en expliciete parameters die je aangeeft bij een functie?
            Expliciete parameters zijn vooraf gedefinieerd, dit is bij rest parameters niet het geval.
